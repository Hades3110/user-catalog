# User Catalog (React + TypeScript)

User catalog page with name search, pagination, and user cards.

Data is loaded from the public DummyJSON API: [DummyJSON - Users](https://dummyjson.com/docs/users).

## Run

1. Install dependencies:
   - `npm install`
2. Start the dev server:
   - `npm run dev`
3. Open in the browser:
   - `http://localhost:5173`

## Features

- Users list with basic info (name, email, avatar, etc.)
- Search by name
- Pagination
- React + TypeScript (no backend, no authentication)
